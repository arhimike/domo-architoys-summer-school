# DØMO ARCHITOYS — сайт и редакционная панель

Сайт дневника летней школы Михаила Корси для архитекторов. Публичная часть собирается Eleventy, а материалы редактируются в браузере через Decap CMS по адресу `/admin/`.

## Что уже подготовлено

- главная страница с архивом из десяти дней;
- развёрнутые отчёты первого и второго дней;
- дни 03–10 заведены как неопубликованные черновики;
- второй день посвящён самостоятельной работе, взаимному тестированию и пошаговым маршрутам пяти инструментов;
- у проекта Розы Archi Helper опубликованы ссылки на первый и детализируемый варианты;
- фотографии первого дня временно полностью убраны из публикации;
- тексты, проекты, итоги, короткий пост, обложка и галерея доступны в редакторе;
- фотографии можно загружать, удалять, переставлять перетаскиванием и временно скрывать переключателем;
- проект 01 подписан «Коллективная идея»;
- проект 05 первого дня: автор — Роза, первоначальное рабочее название — «Метр на метр»;
- ссылка на Telegram-бот регистрации вынесена в настройки сайта;
- видео `IMG_2707.MOV` не включено в публикацию.

## Как редактор будет работать

После подключения GitHub и Netlify откройте:

```text
https://architoys.korsi.ru/admin/
```

В разделе «Дни школы» выберите нужный день. Кнопка «Сохранить» фиксирует изменения в GitHub и запускает автоматическую сборку Netlify. Обычно новая версия сайта появляется через несколько минут.

Пока переключатель «Опубликован» выключен, день остаётся в админке как черновик, а на главной отображается карточка «Скоро». Перед публикацией заполните дату, заголовок, лид и при необходимости обложку.

В списке «Фотографии»:

1. нажмите «Добавить» и загрузите JPG или PNG;
2. заполните описание и подпись;
3. выберите размер карточки;
4. перетащите карточки в нужном порядке;
5. используйте «Показывать», чтобы временно убрать кадр без удаления файла.

Лимит одного загружаемого файла сейчас установлен в 4 МБ.

## Однократное включение админки

Админка использует прямой GitHub backend. Git Gateway не применяется, поскольку он помечен Netlify как устаревший.

Исходники и админка настроены на репозиторий `arhimike/domo-architoys-summer-school`, ветка `main`.

1. В существующем проекте Netlify `summer-school-2026` подключите `arhimike/domo-architoys-summer-school` как Git source. Параметры сборки уже заданы в `netlify.toml`: команда `npm run build`, папка публикации `_site`.
2. Создайте в GitHub OAuth App с callback URL:

   ```text
   https://api.netlify.com/auth/done
   ```

3. Добавьте Client ID и Client Secret в Netlify: **Project configuration → Access & security → OAuth → Install provider → GitHub**.
4. Откройте `/admin/` и войдите через GitHub. У редактора должен быть push-доступ к репозиторию.

Поддомен `architoys.korsi.ru` уже привязан к проекту `summer-school-2026`, поэтому DNS-записи для включения админки менять не нужно.

## Локальный запуск

```bash
npm ci
npm run dev
```

В отдельном терминале для локальной админки:

```bash
npm run cms:proxy
```

Сайт откроется на `http://localhost:8080/`, редактор — на `http://localhost:8080/admin/`.

## Проверка перед публикацией

```bash
npm run qa
```

Проверка пересобирает сайт, валидирует структуру CMS, ссылки, десять карточек дней, атрибуцию проектов и отсутствие опубликованных фотографий.
